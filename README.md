# Acme

This is a single page app made with the following libs and frameworks:

* ES2015 + Babel
* MarionetteJS
* Gulp
* Browserify
* SASS
* Bootstrap
* Jasmine

## Getting started

* Install:
    * Clone and inside this folder run: `npm install`
* Run:
    * `npm start` — starts project
    * `npm test` — run unit tests
